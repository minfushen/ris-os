
# 第一部分：功能点描述文档 (PRD 核心摘要)

### 1. 指标雷达与异常探照灯 (Monitoring & Discovery)
*   **实时战情看板**：展示进件量、通过率、拒绝率、耗时等核心 KPI，支持同比/环比异动自动标红。
*   **异常探照灯 (Searchlight)**：
    *   **无监督团伙发现**：基于设备指纹、关系图谱自动发现聚集性异常进件。
    *   **幽灵拒绝(误杀)分析**：对比外部多头数据，挖掘被我司拒绝但在行业内表现良好的客户，量化误杀成本。

### 2. 深度归因与数据解剖 (Deep Attribution)
*   **策略拦截漏斗**：展示决策流中每一层（准入、反欺诈、信用、评分）的拦截损耗。
*   **数据解剖台 (Data Anatomy)**：
    *   **特征血缘溯源**：点击规则即可看到底层的特征加工逻辑及数据源头。
    *   **特征稳定性(PSI)监控**：实时监控特征分布变化，判定异动是“规则逻辑错”还是“输入数据脏”。

### 3. 策略 Git 流水线 (Strategy Workbench)
*   **可视化策略编辑器**：支持拖拽式决策树、评分卡配置。
*   **离线仿真回测 (Simulation)**：在历史样本上预览新策略效果，产出拦截对比报告。
*   **策略 Git 治理**：
    *   **代码级 Diff 对比**：上线前自动高亮显示策略节点的增删改。
    *   **电子审批流**：集成多级风险会签。

### 4. 上线护盾与一致性观察 (O2O Guardrail)
*   **冠军/挑战者 (A/B Test)**：支持小流量并行实验。
*   **线上线下一致性 (O2O) 检查**：自动比对离线回测值与线上运行值，偏差超限自动预警。
*   **自动熔断机制**：当线上指标触发极端阈值时，毫秒级切换至安全版本。

### 5. 标注飞轮与人机共智 (Labeling Flywheel)
*   **专家质检调查**：人工对争议案例进行定性标注。
*   **知识回流**：标注后的黑/白样本自动打入特征库，触发模型自动增量训练。
